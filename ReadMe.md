# 🐶🐱 Cats vs Dogs Image Classification using CNN

## 📌 Project Overview

This project is a Deep Learning-based image classification system that classifies images of **cats 🐱 and dogs 🐶** using a **Convolutional Neural Network (CNN)**.

The model learns patterns from images and predicts whether the image belongs to a cat or a dog.

---

## 🚀 Features

* Image classification using CNN
* Data preprocessing and normalization
* Model training and evaluation
* Accuracy visualization (graphs)

## 🧠 Tech Stack

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

---

## 📂 Dataset

* Dataset used: Cats vs Dogs dataset
* Source: Kaggle / Online dataset

---

## ⚙️ Model Architecture

* Convolution Layers
* Max Pooling Layers
* Flatten Layer
* Fully Connected Layers
* Activation: ReLU, Sigmoid

---

## 📊 Results

* Achieved good accuracy on validation data
* Model successfully classifies unseen images

## 📌 Future Improvements

* Use Transfer Learning (ResNet, VGG16)
* Improve accuracy with data augmentation
* Deploy using Flask/Streamlit
